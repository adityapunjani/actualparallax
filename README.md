actualparallax
==============

Cross browser Parallax Plugin to turn a set of images / text sections into a smooth and fast parallax scrolling page. 


###Dependancies  
jQuery 1.7.2+

###Options (Defaults)

    imageAttribute: 'data-image',
    container: $('body'),
    speed: 0.3,
    coverRatio: 0.85,
    mediaWidth: 1600,
    mediaHeight: 900,
    parallax: true

###Usage
```
$('.img-parallax').parallaxScroll({
container: $('.parallax-unit'),
});    
```
